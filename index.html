Taller #1: Análisis lexico
  • Presentado Por: Eyleen Polo, Kevin Bolaño
Enunciado

Dado un archivo que contiene información sobre la accidentalidad vial en el país, construya una expresión regular que permita filtar los datos de un periodo particular, donde existan victimas del genero femenino.

Nota: Puede haber varias líneas.

El formato es el siguiente:

YYYY-mm-dd ; placa ; Tipo_Vehiculo ; Nombre ; Edad ; gravedad [fatal | No Fatal] ; Genero [ Male | Female ]

Discriminar los espacios.

El archivo permitido debe ser en formato Texto (.txt)
